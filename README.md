# 蜃龙体育 - 长跑记圈系统

一部手机替代专业电记系统，让每一圈都被准确记录。

## 背景

在广大中小学及基层体育赛事中，长跑比赛的记圈和计时工作一直是个难题：

- **设备落后**：专业电记系统动辄数万元，多数学校无力承担，长跑记圈仍依赖人工手写记录
- **容易出错**：多名运动员同时在场，人工数圈极易漏记、错记，圈数纠纷频发
- **计时不准**：手工秒表只能记录有限名次，无法为每位运动员精确计时
- **排名混乱**：比赛结束后需人工整理成绩，耗时长且容易出错
- **成绩难存**：纸质记录易丢失、难分享，无法快速生成规范成绩单

## 功能特性

### 精准记圈
- 一键为所有运动员记圈，也可单独点击号码快速记圈
- 误记可减圈，退赛可标记，数据实时同步

### 自动计时排名
- 毫秒级秒表自动运行，每位运动员独立计时
- 实时排名动态更新，完赛即出成绩

### 智能语音播报
- 语音播报记圈、最后一圈提醒、完赛通知
- 本地TTS优先，不可用时自动降级为网络TTS（有道/百度）
- 提示音 + 震动双重反馈，三个独立开关可自由控制

### 专业成绩导出
- 一键生成PDF成绩单，含比赛信息、名次、用时
- 弃权/退赛运动员完整标注
- 成绩单可直接打印或一键分享

### 零门槛使用
- 无需专业设备，无需培训，打开APP即可开始
- 支持手动添加或批量导入运动员（文本/文件）
- 比赛设置自动计算总圈数

## 技术栈

- **语言**：Kotlin
- **UI框架**：Jetpack Compose + Material 3
- **架构**：MVVM + StateFlow
- **导航**：Navigation Compose
- **PDF生成**：Android PdfDocument
- **语音**：Android TTS + 网络TTS降级方案
- **提示音**：AudioTrack 程序生成（不依赖TTS引擎）
- **最低SDK**：Android 7.0 (API 24)
